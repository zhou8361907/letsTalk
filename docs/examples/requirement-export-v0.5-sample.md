## 需求 1 · 复制前校验方案状态，停用的方案不允许复制

**状态**：🟢 信息较完整

**在哪个页面**  
数据接入配置页

**改哪里**  
每行/每卡上的「复制此方案」图标按钮

**现在怎样**  
点击复制图标后直接弹出填标识和名称的对话框，不校验方案当前是否启用

**希望改成**  
1）点击复制图标，先校验方案是否处于「启用」状态；  
2）若为停用，弹出提示「该方案已停用，无法复制」，阻止操作；  
3）若为启用，弹窗确认「是否真的要复制」；  
4）确认后弹出填标识/名称对话框；  
5）复制成功后，再弹窗询问「是否删除原来的方案」

**怎么验收**  
1. 点击停用方案的复制图标 → 弹出提示「该方案已停用，无法复制」，无后续弹窗；  
2. 点击启用方案的复制图标 → 先弹「是否真的要复制」→ 确认 → 弹出填标识/名称对话框 → 复制成功 → 再弹「是否删除原来的方案」
