- smip-8.31 模块 = 各省份医保个性化需求；所有省份定制代码均在此模块下，按省份名分包（如 xinjiang/、guangxi/、neimeng/ 等）隔离。
- xinjiang（新疆自治区）和 xjbingtuan（新疆兵团）是两个独立主体，代码路径不同，不能混用。
- 省份定制通常对 smr（支付管理）/ smrts（支付计划）/ smsbm（上报报表）等子模块进行重写或扩展。
- 查询某省特定逻辑时优先看 smip 下对应省份包，再回基线（smr、smc 等）确认通用逻辑。
